# MovieBay Frontend

This is the Vite/React frontend for MovieBay.

## Commands

```sh
npm install
npm run dev
npm run build
```

## Main Folders

- `src/` contains the React app.
- `public/` contains static icons, manifest, robots, and service worker files.
- `supabase/` contains Supabase Edge Functions and migrations used by the app.

Deploy this app with Vercel from this folder.
